        {"href": "", "capabilities": {}, "pool_group": ""},

Returns a JSON object for a single flavor entry:

{"pool": "", capabilities: {...}}
Returns:HTTP | [200, 404]
on_patch(*args, **kwargs)¶

Allows one to update a flavors’s pool_group.

This method expects the user to submit a JSON object containing ‘pool_group’. If none is found, the request is flagged as bad. There is also strict format checking through the use of jsonschema. Appropriate errors are returned in each case for badly formatted input.

Returns:HTTP | [200, 400]
on_put(*args, **kwargs)¶

Registers a new flavor. Expects the following input:

{"pool_group": "my-pool-group", "capabilities": {}}

A capabilities object may also be provided.

Returns:HTTP | [201, 400]
