Welcome to Kolla’s documentation!¶

Kolla’s mission is to provide production-ready containers and deployment tools for operating OpenStack clouds.

This documentation is for the Kolla container images. The following subprojects are available to help deploy Kolla:

  • kolla-ansible
  • kolla-kubernetes

Site Notes¶

This documentation is continually updated and may not represent the state of the project at any specific prior release. To access documentation for a previous release of kolla, append the OpenStack release name to the URL, for example:

http://docs.openstack.org/developer/kolla/mitaka/
