The cinder.volume.drivers.dell_emc.vmax.fast Module¶

class VMAXFast(prtcl)¶

Bases: object

FAST Class for SMI-S based EMC volume drivers.

This FAST class is for EMC volume drivers based on SMI-S. It supports VMAX arrays.
