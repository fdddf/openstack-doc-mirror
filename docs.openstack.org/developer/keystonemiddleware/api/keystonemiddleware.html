keystonemiddleware.s3_token module¶

S3 Token Middleware.

This WSGI component:

  • Gets a request from the swift3 middleware with an S3 Authorization access key.
  • Validates s3 token in Keystone.
  • Transforms the account name to AUTH_%(tenant_name).
