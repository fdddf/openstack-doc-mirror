Nagios/Icinga supportÂ¶

Nagios and Icinga has support for Gnocchi through a Gnocchi-nagios service. It can be installed with pip:

pip install gnocchi-nagios

Source and Documentation are also available.

Gnocchi-nagios collects perfdata files generated by Nagios or Icinga; transforms them into Gnocchi resources, metrics and measures format; and publishes them to the Gnocchi REST API.
