Installation workflow¶

This diagram shows the general workflow associated with an OpenStack-Ansible (OSA) installation.

Figure 1.7. Installation workflow

  1. Prepare deployment hosts
  2. Prepare target hosts
  3. Configure deployment
  4. Run foundation playbooks
  5. Run infrastructure playbooks
  6. Run OpenStack playbooks
